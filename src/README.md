# 2017-mn-group-12


## How to Run

This application can be started directly by putting up the following code.
```
mvn clean install
```

After this, visit - 

```
http://localhost:4567/
```
### Test Cases
Unit tests have been written. Test cases can be checked by compiling the code.

### Required Technologies
* Maven
* Java 8
* Free Marker Template (.ftl)
* JavaScript
